# DogBreedExploration
This is a fun project to track what dogs have I seen

## Features
- The app can store the name of the dog/cat breed via typing or voice
- It can save the images also If someone wants to upload
- It can create a heat map with the dog breed icons and show me the cluster
- Maybe can create a dog/cat breed hunt for the user with all the world known breed and can tick mark the one user already seen it
- It can suggest the breed name based on the user query on height, hair color, hair type, eyes or tail etc
